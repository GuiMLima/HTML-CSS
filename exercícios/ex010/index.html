Usando links externos

Como funciona um link?

Um link serve para abrirmos uma nova página com o intuito de encontrarmos algo que procuramos através do navegador.

A tag que usamos para inserir um link é a tag A.

Além do uso dessa tag, também usamos alguns atributos para expecificarmos "como" ele seja aberto.

Esses atributos são:

  1. TARGET: Como ele vai abrir, se vai abrir por cima da outra janela, se vai abrir outra janela, etc:
    • _blank: abre em uma nova janela em branco;
    • _self: abre na janela atual;
    • _top: desfaz todos os frames e abre no navegador completo;
    • _parent: similar com o anterior em uma referência à janela mãe.
  2. REL: indicar a natureza do destino:
    • next: próxima parte do documento;
    • prev: parte anterior do documento;
    • author: link para o site do autor do artigo;
    • external: link para outro site que não faz paarte do site autal;
    • nofollow: lin pago.
