Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag B (não semântica).

Nesta frase, tenho um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica).

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

Texto marcado

É possível marcar algummas palavras com a tag MARK.

Texto grande e pequeno

Também é possível modificar o tamanho das letras, usando duas tags, a primeira para indicar uma palavra grande, usando a tag BIG

. Porém, a essa tag não é usada mais, então não é masi certo usa- la. Também temos a tag SMALL.

Texto deletado

Para deletarmos um texto dessa forma, usamos a tag DEL.

Texto inserido

Podemos marcar com a tag INS também.

Texto sobrescrito

O número fica elevado, por exemplo: x20+3

Para isso, usamos a tag SUP.

Texto subscrito

O número fica por baixo, por exemplo: H2O

Para isso, usamos a tag SUB.
