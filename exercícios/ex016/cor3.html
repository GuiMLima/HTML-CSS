Círculo cromático

O que é um círculo-cromático?

Um conjunto de milhares de cores diferentes, em resumo. Temos diferentes classes de cores dentro desse círculo cromático, sendo essas classes diferentes:

  • Cores primárias:
    • Amarelo;
    • Azul;
    • Vermelho.
  • Cores secundárias:
    • Laranja;
    • Roxo;
    • Verde.
  • Cores terceárias:
    • Amareo-esverdeado;
    • Amarelo-alaranjado;
    • Vermelho-alaranjado;
    • Vermelho-arroxeado;
    • Azul-arroxeado;
    • Azul-esverdeado.

Temperatura das cores

Cores frias

Nos dão uma sensação mais tranquila, calma, sendo elas:

  • Amarelo-esverdeado;
  • Verde;
  • Azul-esverdeado;
  • Azul;
  • Azul-arroxeado;
  • Roxo.

Cores quentes

Diferente das cores frias, as quentes são mais agitadas, mais movimentadas:

  • Amarelo-alaranjado;
  • Vermelho;
  • Amarelo;
  • Vermelho-alaranjado;
  • Vermelho-arroxeado;
  • Laranja.
