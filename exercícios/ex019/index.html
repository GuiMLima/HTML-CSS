Identificador

id

Dentro do HTML, o Identificador é "id", mas no CSS, ele é representado como "#".

Importante usa- lo para caso o desenvolvedor queira alterar o primeiro títuo do site mas os outros permaneçam do jeito que está.

class

Dentro do HTML, o Identificador personalisado é "class", mas no CSS, ele é representado como ".".

Diferença entre "id" e "class"

A diferença é simples, id é usado apenas uma vez, obtendo um classe nomeada "principal", enquanto class é possível criar diferentes classes com nomes diferentes.

Evite nomear seletores com nome de cores, pois é possível que você precise mudar o nome e fica estranho em seguida.
