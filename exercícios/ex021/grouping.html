Grouping tags e semantic tags

A linguagem HTML possui duas tags de agrupamento genéricos:

  • Div: blockline-level;
  • Span: inline-level.

    • Com o surgimento do HTML5, essas tags se tronaram não-semânticas, pois agora temos tags para dividir partes do documento HTML:

  • Header: cria áreas relativas ao cabeçalho;
  • Nav: define áras que possui os links de navegação pela estrutura de páginas do website. Um nav pode estar dentro de um header;
  • Main: agrupador que delimita o conteúdo principal do site;
  • Section: cria seções para a página. Pode conter o conteúdo diretamente do corpo ou dividir em artigos com conteúdos específicos;
  • Article: elemento que vai conter um conteúdo que pode ser lido de forma independente e dizem respeito do mesmo assunto. podemos usar esse elemento para delimitar um post de blog ou fórum;
  • Aside: está posicionado ao lado de um texto ou no meio dele;
  • footer: possui informações sobre a autoria do conteúdo.
